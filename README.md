# 全自动提交友链系统
Friend's website links auto submit system


## 提交步骤

1. 首先在自己的网站友链页面中，添加我的友链信息；
2. 按照模板在 Issue 中提交即可；
3. 提交成功后，在主页：https://extingstudio.com/friends/ 刷新查看 (未显示的话请检查issue格式，并稍等片刻，刷新页面)；

> 
> [点击进入友链主页](https://extingstudio.com/friends/)
>
